# Change Log

All notable changes to the "gams" extension will be documented in this file.

## [0.0.7] - 2025-30-20

### Modified

* Syntax highlighting for plural/singular form of field names in MPSGE

## [0.0.6] - 2025-01-15

### Modified

* Syntax highlighting fixed (variables)

## [0.0.5] - 2024-11-21

### Added

* Button in the status bar to open GAMS extension settings. Clicking the button navigates directly to the extension's settings page.
* Button in the status bar to toggle the `runProject` setting. The button displays the current status (`On`/`Off`) and allows users to toggle it with a single click.

## [0.0.4] - 2024-06-07

### Added

* Command line parameter can be added

## [0.0.3] - 2024-05-20

### Added

* Open GDX button and "Open GDX with GAMS IDE" menu in the explorer

### Modified

- Syntax highlighting fixed

## [0.0.2] - 2024-05-06

### Added

- Support running project batch file.
- Icons for GAMS related files.

## [0.0.1] - 2024-04-01

### Added

- Initial release of the extension.
